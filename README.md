# ZebraScienceFrontEnd
ZebraScienceFrontEnd-Vue.js

## Install

```bush
// install dependencies
cnpm install
```

## Run

### Development

```bush
// For the first time, run init to create index.html
npm run init
npm run dev
```


```
//If you can run the programme, you must run the following command
```
